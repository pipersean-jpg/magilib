Cloudinary setup (free high-res storage)

    

      
1. Sign up free at cloudinary.com (no credit card)

      
2. From your dashboard, copy your Cloud name

      
3. Go to Settings → Upload → Add upload preset

      
4. Set Signing mode to Unsigned, save, copy the preset name

      
5. Paste both into the fields above

      
Once configured, every new cover photo is automatically stored in full resolution. The catalog still loads fast using the compressed thumbnail, but tapping zoom shows the full-res version.
